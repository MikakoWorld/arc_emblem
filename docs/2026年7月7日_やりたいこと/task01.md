# 追加要件定義: 土台比率・エンブレム効果・スマホUI改善

## 目的

`index.html` のカスタマイズ機能を拡張し、正方形の土台作成、エンブレム本体への装飾効果、スマホでの操作性改善に対応する。

## 対象ページ

- `/Users/mikako/アーク/custom_emblem/index.html`

## 追加要件

### 1. 土台比率に `1:1` を追加

土台の比率選択に `1:1` を追加する。

サイズごとの推奨ピクセル数は以下とする。

| サイズ | 1:1 |
| --- | --- |
| S | 1024 x 1024 |
| M | 1600 x 1600 |
| L | 2400 x 2400 |
| XL | 3200 x 3200 |

既存の `3:4` / `9:16` と同じように、プレビュー、PNG保存、SVG保存、iPhone保存に反映する。

### 2. 「その他効果」設定を追加

設定欄に「その他効果」枠を追加する。

この枠では、エンブレム本体に対して以下の効果を任意で付けられるようにする。

- シャドウ
- フチ線

効果はプレビューだけでなく、PNG保存、SVG保存、iPhone保存のすべてに反映する。

### 3. シャドウ設定

シャドウはチェックボックスまたはトグルで ON/OFF できるようにする。

ON の場合、以下を設定できるようにする。

| 項目 | UI | 初期値 | 備考 |
| --- | --- | --- | --- |
| 色 | カラーピッカー | `#000000` | シャドウの色 |
| 透明度 | スライダー | `35%` | 0% から 100% |
| ぼかし | スライダー | `12px` | 旧メモの「範囲」に相当 |
| 距離 | スライダー | `8px` | エンブレムからの離れ具合 |
| 角度 | スライダー | `135°` | 0° から 360° |

角度と距離からシャドウの X/Y オフセットを計算する。

### 4. フチ線設定

フチ線はチェックボックスまたはトグルで ON/OFF できるようにする。

ON の場合、以下を設定できるようにする。

| 項目 | UI | 初期値 | 備考 |
| --- | --- | --- | --- |
| 色 | カラーピッカー | `#ffffff` | フチ線の色 |
| 透明度 | スライダー | `100%` | 0% から 100% |
| 太さ | スライダー | `12px` | 書き出しサイズに応じて自然に見える太さにする |

実装では、エンブレム本体を下に複製して太い線を付け、その上に通常のエンブレムを重ねる方式を推奨する。

理由:

- 元SVG内の複数パスに直接 `stroke` を付けるより、外側のフチとして見えやすい
- PNG保存とSVG保存の見た目を揃えやすい
- 既存の単色・グラデーション塗りと併用しやすい

### 5. スマホ表示のアコーディオン化

スマホ表示では、設定欄が縦に長くなりすぎないように以下の枠をアコーディオン化する。

- エンブレム
- 土台
- その他効果

開閉操作は見出し右側の `+` / `-` アイコンをタップして行う。

初期状態は以下とする。

| 枠 | 初期状態 |
| --- | --- |
| エンブレム | 閉じる |
| 土台 | 閉じる |
| その他効果 | 開く |

PC表示では既存と同じように、設定を常に表示してよい。

## 期待する挙動

- `1:1` を選ぶと、プレビュー枠と保存サイズが正方形になる
- シャドウ ON のとき、エンブレム本体に影が付く
- フチ線 ON のとき、エンブレム本体の外側に縁取りが付く
- シャドウとフチ線は同時に使える
- 背景色、背景透明度、余白、カラー方式と組み合わせても破綻しない
- PNG保存、SVG保存、iPhone保存でプレビューに近い見た目が出力される
- スマホでは設定欄のスクロール量が減り、必要な枠だけ開いて操作できる

## 実装メモ

- 効果はエンブレム本体に付ける。土台背景には付けない。
- シャドウはSVGフィルターで表現すると、SVG保存にも反映しやすい。
- PNG保存は現在のSVGをCanvasに描画しているため、SVG側に効果を含める実装が望ましい。
- フチ線の太さは固定pxのままだとサイズ差で見え方が変わるため、書き出しサイズまたはSVG viewBox基準で調整する。
- アコーディオンはアクセシビリティのため、ボタン要素と `aria-expanded` を使う。
